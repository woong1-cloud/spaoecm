1. 상품분석판매 데이터 (필수)

📁 파일 형식: CSV (.csv)
📋 포함 내용: 상품명, 옵션, 현재재고, 판매수량
🎯 결과: SKU 생성 + 판매량 분석
예: 20260129_091301_ProductPrdchart.csv

2. 물류센터1 재고 데이터 (선택)

📁 파일 형식: Excel (.xlsx, .xls, .xlsb)
📋 포함 내용: 상품(SKU), 솔리드가용재고, 아소트가용재고
🎯 결과: 물류센터1 재고 추가 (SKU 매칭)
예: test0129_center1.xlsx

3. 물류센터2 재고 데이터 (선택)

📁 파일 형식: Excel (.xlsx, .xls, .xlsb)
📋 포함 내용: 상품(SKU), 솔리드가용재고, 아소트가용재고
🎯 결과: 물류센터2 재고 추가 (SKU 매칭)
예: test0129_center2.xlsx

4. 매장 재고 데이터 (선택)

📁 파일 형식: Excel (.xlsx, .xls, .xlsb)
📋 포함 내용: 상품코드(SKU), 가용재고
🎯 결과: 매장 가용재고 추가 (SKU 매칭)
예: T009_10_CHANNEL_AVAILABLE_STOCK_20260203105740.xlsx

5. 분배내역 (선택)

📁 파일 형식: Excel (.xlsx, .xls, .xlsb)
📋 포함 내용: SKU(또는 상품코드), 분배량(또는 N열/수량) — 분배량 기준으로 수량 표시
🎯 결과: 대시보드 테이블 분배내역 칼럼에 수량 표시 (필업제안 ↔ 분배가능 사이)
예: 분배요청(1).xlsx — SKU·분배량(또는 N열) 컬럼 포함 시 같은 SKU는 수량 합계 표시

6. 옴니판매불가 SKU (선택)

📁 파일 형식: Excel (.xlsx, .xls, .xlsb)
📋 열 구성(기본 가정): C열=매장명, D열=스타일코드, E열=단품코드(SKU), H열=판매불가 수량
🎯 결과: 대시보드 하단 옴니판매불가 SKU 섹션에 스코어보드·상세 테이블로 표시
예: 옴니채널불가상품_YYYYMMDDHHMMSS.xls(x)
